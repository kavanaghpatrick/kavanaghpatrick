## GPU & Systems

- :zap: **[gpu-native-os](https://github.com/kavanaghpatrick/gpu-native-os)** -- GPU-native operating system for Apple M4. All application logic runs entirely on the GPU with CPU as I/O coprocessor. Metal compute shaders, persistent kernels, GPU-native event system and text renderer.
- :robot: **[metal-attention](https://github.com/kavanaghpatrick/metal-attention)** -- Composable attention kernel inference engine for hybrid AI models (Jamba, Griffin, RWKV, Zamba) on Apple Silicon. Trait-based attention with zero-overhead specialization via Metal. Linear attention at ~0.12x wall-clock vs softmax.
- :dna: **[emojiasm](https://github.com/kavanaghpatrick/emojiasm)** -- Assembly language made of pure emoji. Stack-based VM with 37 opcodes, AOT compiler to native (~250x), and GPU execution via Metal compute kernels. 4,700x speedup with 10K parallel GPU instances. First bytecode interpreter on Apple Metal.

## Formal Verification

- :balance_scale: **[us-tax-code-lean](https://github.com/kavanaghpatrick/us-tax-code-lean)** -- The entire US Internal Revenue Code formalized in Lean 4. 765 IRC sections, 537K lines, 607 proven theorems, 93% income tax coverage. Loophole detection and contradiction analysis.
- :pill: **[aristotle_biochemistry](https://github.com/kavanaghpatrick/aristotle_biochemistry)** -- Formal verification for drug safety. Proves mathematical impossibility of hERG cardiac toxicity binding using Lean 4 and geometric proofs. 0% false negative rate, 43.2% coverage via pure math.
- :triangular_ruler: **[aristotle-math-problems](https://github.com/kavanaghpatrick/aristotle-math-problems)** -- Automated harness for solving open mathematical problems. States bare conjectures, submits to Aristotle theorem prover with compounding context. Multi-AI support (Grok, Gemini, Codex, Claude).
